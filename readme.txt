/* readme.txt */

Έφτιαξα ένα σκελετό με όλες τις κλάσεις που θα μου είναι χρήσιμες για το δικό μου κομμάτι.
Και έκανα και όλο το linking ώστε να είναι ο κώδικας σε αυτή την πρώτη του έκδοση εκτελέσιμος.

(Κοινά) Αρχεία που ίσως θες να ρίξεις μια ματιά :

Στο params.hpp, έχω δηλώσει μερικές global παραμέτρους του αλγορίθμου, για ευκολία, αντί να τις περνάω κάθε φορά
σαν τρόμπας στα ορίσματα του κάθε constructor. Αυτές οι παράμετροι παίρνουν τιμή στην main της lsh.cpp
΄Ισως θες να κάνεις και συ κάτι τέτοιο για πιθανές δικές σου παραμέτρους για το cube.


Τα input_check.cpp/.hpp περιέχουν συναρτήσεις που τσεκάρουν τα arguments από το command line και αρχικοποιούν τις παραμέτρους
του αλγορίθμου, και συναρτήσεις που διαβάζουν τα αρχεία δεδομένων που μας έδωσαν.
Αυτές οι συναρτήσεις μπορουν να κληθουν και απο την δικη σου main για το cube για τους ίδιους σκοπούς.

Τα dataset.cpp/.hpp περιέχουν την κλάση που αποθηκεύει ουσιαστικά (δείκτες στα) σημεία που διαβάζω σαν είσοδο από
τα αρχεία που μας έδωσαν, είτε τα σημεία εισόδου, είτε τα query.
Εκεί μπορείς να αποθηκεύεις και συ τα σημεία από τα αρχεία που διαβάζεις (τα ίδια αρχεία είναι και για το cube και για το Lsh ούτως ή άλλως)

Τα object.cpp/.hpp περιέχουν την κλάση που ορίζει και περιγράφει ένα d-σημείο-αντικείμενο.
Απλά έχω ένα πίνακα μεγέθους d (δυναμικό) που κρατάει τις συντεταγμένες του σημείου, + ένα ID
που χρησιμεύει αργότερα στην εύρεση του nearest neighbor.  Το ID εσύ μπορείς να το αγνοήσεις, δεν σου χρησιμεύει κάπου νμζω.


Τα h_hash.cpp/.hpp περιέχουν την κλάση που ορίζει γενικά το H family συναρτήσεων.
Οπότε πολύ απλά μπορείς να ορίσεις μια h function ως ένα αντικείμενο αυτής της κλάσης.
Επίσης έχω κάνει overload τον operator (), ώστε να μπορείς να χρησιμοποιήσεις ένα αντικείμενο αυτής της κλάσης
ως συνάρτηση. Δηλαδή, έχοντας ορίσει ένα αντικείμενο έστω h της κλάσης,
να μπορείς , δεδομένου ενός σημείου αντικειμένου Object p, να κάνεις h(p) και να σου επιστραφεί η τιμή της συνάρτησης όπως ορίζεται
στις διαφάνειες με το floor .

Στη main στο lsh.cpp μπορείς να κοιτάξεις πως καλώ τις συναρτήσεις που τσεκάρουν τη γραμμή εντολών, που διαβάζουν το αρχείο εισόδου και που δημιουργουν
ένα αντικείμενο τύπου Dataset που θα αποθηκεύει δείκτες στα δημιουργημένα σημεία.  Κάτι αντίστοιχο μπορείς να κάνεις και συ στη δική σου main για το cube.cpp
